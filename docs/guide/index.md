# 📘 使用指南
  
在加入前，请先了解 **公费服** 与 **公益服** 的定位差异，选择适合自己的使用方式。

---

## 💳 公费服 vs 🌱 公益服

### 💳 公费服
- 面向 **长期、稳定使用** 的用户
- 服务与资源相对固定，使用预期清晰
- 有明确的使用规则与维护说明
- 适合希望获得 **持续、可预期体验** 的用户

👉 [查看公费服说明](paid/)

---

### 🌱 公益服（活动抽取账号）
- 以 **活动抽取账号或随机销售账号** 的形式提供
- 主要用于体验或参与特定活动
- 账号具有 **临时性与不确定性**
- 使用规则较为严格，可能随活动结束而回收或调整

👉 [查看公益服说明](public/)

---

> 建议首次使用的用户 **先确认账号类型**，  
> 并完整阅读对应说明页，避免因误解规则导致账号异常。
