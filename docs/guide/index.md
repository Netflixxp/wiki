# 📘 使用指南
  
在加入前，请先了解 **公费服** 与 **公益服** 的定位差异，选择适合自己的使用方式。

---

## 💳 公费服
- 面向 **长期、稳定使用** 的付费用户
- 服务与资源相对固定，使用预期清晰
- 有明确的使用规则与维护说明
- 适合希望获得 **持续、可预期体验** 的用户

👉 [查看公费服说明](paid/)

---

## 🌱 公益服
- 以 **活动抽取或随机销售** 的形式提供
- 需要签到保号
- 账号具有 **临时性与不确定性**
- 资源相对公益服要慢和少一些

👉 [查看公益服说明](public/)

---

> 建议首次使用的用户 **先确认账号类型**，  
> 并完整阅读对应说明页，避免因误解规则导致账号异常。
